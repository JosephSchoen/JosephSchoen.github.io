Optum:

During my internship with Optum this past summer, 2022, I had the privillage of working with the Microsoft platform. My team was given the job of putting together a system to take information from a Microsoft SharePoint list, to a SQL table in the Optum database. To do this, the team used Power Automate, a Microsoft application that allows connections between Microsoft softwares to be done automatically. By using Power Automate, the team was able to draw all the list information in the Microsoft SharePoint, then copy that information to an SQL table in the database. It happened automatically, which means when there was an update to the Microsoft SharePoint, the SQL table would also get updated, mimicing the Sharepoint list perfectly. This project allowed the active Microsoft Sharepoint list to exist in an SQL table, that way other services that create reports can pull the SQL table information, and render the report with the new information. This was not possible to gather from Microsoft Sharepoint automatically.

CISC 420 - Computer Graphics:

In my second year of college, I took computer graphics! The following project, 3D Projections using wireframe drawing, was a fun yet complicated project! In the window below, you are able to move in the scene using W, A, S, and D, as well as rotate the scene with the left and right arrow keys. This project demonstrates live rendering of translations on three dimensional objects.



The project below, Shading and Illumination, focused on shading a scene with the Gouraud shading algorithm and the Phong shading algorithm. In the window below, you are able to move in the scene using W, A, S, and D, as well as rotate the scene with the left and right arrow keys. Additionally, you are able to select how many light sources you would like. Default is 1, max is 10. You are able to change the location of each light source, as well as its color! All of this is done with live rendering of the objects!

By using the buttons below, the scene can be changed to render as black and white, fish eye effect, ripple effect, toonify effect, and a custom effect.
